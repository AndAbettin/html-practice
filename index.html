1. What is the difference between the usage of HTML and CSS?
    • HTML deals with the fundamental creation of elements; it's the skeleton and muscle of a web page in that it provides the structure and motion. CSS is a tool to style HTML's elements; it would be the skin, hair, etc. in that it affects aesthetics.

    • Semantic elements describe the content they contain. Non-semantic elements have no inherent meaning and say nothing about the content they contain.
